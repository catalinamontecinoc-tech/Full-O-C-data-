# Eclipse Timing Variation data

This repository contains the eclipse timing measurements used in the analysis of the PCEB HS 0705+6700 and NSVS 14256825

## Files

- `HS0705_eclipse_times.csv`  
  Columns:
  - Cycle: eclipse cycle number
  - BJD: mid-eclipse time (BJD)
  - Error: timing uncertainty (days)

- `NSVS1425_eclipse_times.csv`  
  Columns:
  - Cycle: eclipse cycle number
  - BJD: mid-eclipse time (BJD)
  - Error: timing uncertainty (days)
